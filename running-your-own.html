Как сгенерить HTML на своем компьютере

Пн. 01 Декабрь 2014
dim

Процедура сборки HTML страниц сайта на своем компьютере очень проста.

ВНИМАНИЕ! Пока что описываемый процесс подходит только для Linux.

Итак, по шагам.

  1. Выписать локально репозиторий:
$ git clone https://github.com/openwrt2ru/openwrt2ru.github.io
  1. Установить pip:
$ sudo apt-get install pip
  1. Зайти в каталог репозитория и собрать virtualenv:
$ make venv
  1. Подредактировать нужные страницы сайта (в ветке content). Разумеется, если хотим просто сгенерить локально HTML то этот шаг можно пропустить.
  2. Запустить локальный сервер:
$ make serve
  1. Открыть http://127.0.0.1:8000 и посмотреть что получилось.
  2. Остановить сервер можно просто нажав <Ctrl>-C в консоли, где он был запущен.
